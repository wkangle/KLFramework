# KLFramework
应用通用基础能力lib-KLFramework

 
 
 
使用：

1.在项目根目录的build.gradle 添加
```groovy
    allprojects {    
        repositories {
             maven { url "https://raw.githubusercontent.com/wkangle/KLFramework/master" }   
              }
    }

```
       
###### 
2.在module中引入
``` groovy
    implementation 'com.kpl:framework:1.0.0'
```
